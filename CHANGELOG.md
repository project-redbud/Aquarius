# Changelog

## v1.1.0 (2026-07-01)

### 新功能
- **精华瓶 💎**：管理员可将瓶子设为精华，获得 5 个虚空点赞 + 【精华】标识，瓶主收到通知
- **小黑板 📋**：首页展示未过期的通知瓶，所有人可见
- **图片双指缩放**：点击放大后支持双指缩放（0.5x~5x）、双击 1x/2x 切换、鼠标滚轮缩放
- **只看我 👁**：评论区新增「只看我」按钮，过滤仅显示自己的评论
- **管理员标识用户名**：管理员评论可显示用户名（需开启个人设置），支持「匿名（🛡️admin）」格式
- **瓶子内容 2000 字**：内容上限从 500 提升至 2000 字符，列表截断统一为 200 字
- **管理面板删评论**：支持单独删除评论/回复，无需删整个瓶子

### 修复
- **/daily 页面 400 卡死**：服务器返回完整 7 天窗口 + 边界，前端纯客户端翻页，消除日期越界
- **管理员用户名显示**：非管理员用户也能看到管理员评论的用户名
- **分享 URL**：App 内分享使用站点设置的 URL 而非 localhost
- **首页热门评论数**：修复 CommentCount 始终为 0（EF Core 未 Include Comments）
- **推送日期逻辑**：不允许查看未来日期推送；今天无推送时默认显示最后有推送的日期
- **定时推送瓶创建时间**：使用目标日期而非管理员操作时间
- **精华赞数即时刷新**：切换精华后赞数立即 ±5
- **评论编辑/删除**：普通用户恢复编辑删除自己评论的能力
- **管理员/瓶主标识**：勾选后不再自动重置

### 优化
- 评论区默认排序改为「最早」
- 通知列表新增 `essence_on/off` 日志翻译
- 精华通知归入 ❤️ 点赞分类
- 推送管理截断统一为 200 字
- 管理面板站点 URL 备注更新
